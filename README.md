🛒 Shopping Website Admin Panel

A powerful Admin Dashboard for an E-commerce / Shopping Website that allows administrators to manage products, orders, and users efficiently.
This project focuses on building a structured admin interface for managing an online store.

📌 Description
The Shopping Admin Panel is designed to help administrators control and manage different aspects of an e-commerce platform. Through this dashboard, admins can add, update, and delete products, track customer orders, and manage user information.
This project demonstrates concepts like CRUD operations, dashboard UI design, and admin-side management systems used in real-world e-commerce applications.

🚀 Features

✅ Admin login system
✅ Add new products
✅ Update product details
✅ Delete products
✅ View all products
✅ Manage orders
✅ View registered users
✅ Dashboard analytics overview
